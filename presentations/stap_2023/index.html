Data Reduction Workflows


  • Eight potential operation modes for ESTIA
  • Three binary options that should be considered in reduction:
          Focusing mode / Polarisation / Pulse skipping
  • Currently workflows exist for the 100 and 000 modes
  • Next 6 months: Developing McStas simulations to cover the remaining modes



Data from OFFSPEC instrument, reduced with Mantid and scipp, the scipp reduced data was then analysed with EasyReflectometry.

EasyReflectometry

EasyReflectometry


  • Easy to use and extensible analysis package for reflectometry
  • Built on successes of EasyScience/EasyDiffraction
  • Functionality includes co-refinement, such as for multiple isotopic contrasts



The sample tab for the EasyReflectometry GUI interface.
