Probabilistic Analysis

Likelihood sampling

  • Some Monte-Carlo approach is taken to sampling the likelihood, \(p(D|\Theta)\)
  • Markov-chain Monte Carlo is the most popular

Bayesian Analysis

Bayes' Theorem


\[ p(\Theta|D) = \frac{p(D|\Theta)p(\Theta)}{p(D)} \propto p(D|\Theta)p(\Theta) \]
doi: 10.1098/rstl.1763.0053

Bayesian Model Selection

\[ p(D|M) = \iint p(\Theta|D)p(\Theta) \;\;\text{d}^N\Theta \]
  • We can estimate this multidimensional integral
  • Nested sampling is one of the most popular methods


doi: 10.1063/1.1835238

Bayes' Factor

\[ 2\ln [B_{21}] = 2\Big\{\ln \big[p(D|M_2)\big] - \ln \big[p(D|M_1)\big]\Big\} = 10.95\pm0.37 \]
\[ 2 \ln [B_{21}] \] Interpretation

\[ [2, 6) \] Positive
\[ [6, 10) \] Strong
\[ [10, \infty) \] Very strong

uravu Investigates Relationships


                from uravu.relationship import Relationship

                modeller = Relationship(function, x, y, ordinate_error=yerr)
                modeller.max_likelihood('mini')
                modeller.mcmc()
                modeller.nested_sampling()
  • Maximum likelihood evaluation uses scipy
  • Markov-chain Monte Carlo uses the emcee package
  • Nested sampling uses the dynesty package

Some Implementation Details

  • uravu can accept three types of ordinate data:
    1. Traditional y and yerr
    2. A list of scipy.stats.rv_continuous objects
    3. A list of uravu.distribution.Distribution objects
  • Internally, uravu uses an \(N\)-dimensional Gaussian kernel density estimation to describe the data
